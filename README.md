# ATOMQUEST GoalSphere

ATOMQUEST GoalSphere is an enterprise Goal Setting & Performance Tracking Portal built for the AtomQuest Hackathon.

The platform helps organizations manage employee goals, approvals, quarterly check-ins, analytics, and performance tracking through role-based dashboards for Employees, Managers, and Admins.

Key Features:
- Goal creation & approval workflow
- Quarterly performance check-ins
- Shared departmental goals
- Analytics dashboards & reports
- Goal cycle management
- Audit logs & notifications
- Performance leaderboard
- Responsive enterprise UI

Tech Stack:
- React
- TypeScript
- Tailwind CSS
- Recharts
- LocalStorage

This project focuses on improving organizational visibility, accountability, and performance management with a modern enterprise dashboard experience.
